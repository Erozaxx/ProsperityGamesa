# Naming Conventions

## Agent slugs
kebab-case (orchestrator, requirements, product-strategist, cto, cio, cfo, architect, finops-domain-expert, challenger, security, creative, learning-designer, content-writer, workshop-facilitator, exercise-designer, prototype, audience-adapter, storyline-crafter, coder, reviewer, tester, process, bfu)

## Soubory
- Markdown: `snake_case.md` nebo prefixy (`brief_`, `handoff_`, `report_`)
- Logs: `YYYY-MM-DD_run.log`
- Decisions: `DR-001_short-topic.md`

## Iteration IDs
`iter-001`, `iter-002`, ... (nebo `iter-001a`, `iter-001b` pro fáze)

## Task IDs
`T-001`, `T-002`, ... (unikátní v rámci iterace, přiřazuje Orchestrátor)
